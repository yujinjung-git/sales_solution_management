# 적용 내용

- 왼쪽 사이드바에 `유지보수 계약관리` 메뉴를 새로 추가했습니다.
- 유지보수 계약관리 화면에 대시보드, 계약관리 목록, 계약 상세, 청구 처리 현황, 제품별 고객관리 영역을 추가했습니다.
- 계약 상세 기본정보에서 계약 기본정보, 계약 제품/수량, 계약 금액을 한 번에 확인하도록 구성했습니다.
- `/docs` 폴더에 GitHub Pages 배포용 빌드 결과를 포함했습니다.

## 배포 설정

GitHub Pages 설정은 아래와 같이 유지하세요.

- Source: Deploy from a branch
- Branch: main
- Folder: /docs
- Custom domain: 비움
