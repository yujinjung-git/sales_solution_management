# SalesOps AI CRM MVP 기획

## 우선 구현 범위

이번 MVP는 영업 실무에서 즉시 가치가 큰 1, 2, 3번 기능을 먼저 검증한다.

1. AI 기반 맞춤형 제안서 및 공식 공문 자동 초안 생성
2. 라이선스 갱신 및 Up-sell/Cross-sell 기회 자동 탐지
3. 과거 Win/Loss 데이터 기반 최적 할인율 가이드

## 핵심 사용자 흐름

영업 담당자는 고객사와 영업 건을 선택한 뒤, 같은 CRM 데이터를 기준으로 문서 초안, 갱신 기회, 가격 전략을 확인한다. 각 기능은 별도 메뉴로 나뉘지만 고객사, 계약, 딜, 과거 사례 데이터는 공유한다.

## Feature 1. AI 문서 생성

### 목적

반복되는 공문, 제안서, 기술 질의 답변서, 견적 구성안을 빠르게 초안화한다.

### 입력 데이터

- 고객사 정보
- 산업군
- 도입 제품 및 모듈
- 라이선스 수량
- 고객 문의 내용
- 이전 미팅/대응 요약
- 유사 성공 사례
- 유사 실패 사례

### 출력

- 공식 문서 제목
- 고객 요구사항 요약
- CRM 근거 기반 답변
- 권장 구성
- 상업 조건
- 후속 액션

### 향후 AI 연동

초기에는 템플릿과 규칙 기반으로 생성하고, 이후 LLM API에 CRM 컨텍스트와 유사 사례 검색 결과를 전달해 초안 품질을 높인다.

## Feature 2. 갱신 및 업셀 탐지

### 목적

라이선스 만료, 유지보수 도래, 사용량 증가, 미도입 모듈을 기준으로 추가 매출 기회를 선제적으로 알려준다.

### 주요 판단 신호

- 라이선스 만료 D-90, D-60, D-30
- 사용량 85% 이상
- 특정 모듈 미도입
- 산업군 내 유사 고객의 추가 도입 사례
- 고객 건강도 하락

### 출력

- 갱신 D-Day
- Churn 위험도
- 예상 추가 매출
- 추천 액션
- 이탈 방지 조건

## Feature 3. 가격 및 할인 가이드

### 목적

담당자의 감이나 기억에 의존하는 가격 협상을 줄이고, 과거 수주/실패 데이터를 기준으로 협상 전략을 제공한다.

### 입력 데이터

- 산업군
- 사용자 수
- 경쟁사 참여 여부
- 요청 할인율
- 서버 이중화 포함 여부
- 유사 Win/Loss 사례

### 출력

- 권장 할인율
- 최대 허용 할인율
- 예상 수주 확률
- 마진 경고
- 가격 방어 논리
- 유사 사례 테이블

## 필요한 데이터 모델

- customers: 고객사 기본 정보
- contacts: 고객 담당자 및 의사결정자
- deals: 영업 기회
- activities: 미팅, 전화, 이메일, 공문 이력
- products: 제품 및 모듈
- licenses: 계약, 라이선스, 유지보수 정보
- quotes: 견적 및 할인 정보
- documents: 생성 문서와 버전
- cases: 유사 성공/실패 사례
- recommendations: AI 추천 액션
